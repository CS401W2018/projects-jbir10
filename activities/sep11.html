Internet

The Internet is a collection of computer networks and electronic devices that are connected.

Web

The web is a system where information is shared on the internet through web browsers.

How are the Internet and Web different?

The internet is a collection of networks while the web is a collections of websites shared on the internet.

ASCII and Unicode

ASCII is an encoding standard for communication between computers. Unicode is encoding standard for various different languages.

What is UTF-8

UTF-8 is an encoding system for Unicode that translates between unicode character and any binary string

URL

URL is the address of a web page

Domain Name

Domain name is a name of the Website for example google.com

HTTP

Hypertext Transfer Protocol, a set of standards that allows data transfer on the web.

HTTPS

Hypertext Trasfer Protocol Secure, a more secure http

IP Address

ip address is a unique set of number that is autimatically assigned when a device connects to a network, it helps identify and locate devices on the network.

DNS

Domain name System basically translates domain names to ip addresses to load webpages.

W3C Recommendations

are guidelines and standards established by the World Wide Web Consortium to guarantee accessibility and compatibility for developers and organizations on the web.

XML, XHTML, HTML, HTML5

XML (Extensible Markup Language) is a markup up language that offers guidelines for defining any type of data.

XHTML(Extensible Hypertext Markup Language) is a more XML-based and strict version of html.

HTML(Hypertext Markup Language) is a markup language that is used to create Web pages.

HTML5 is the latest version HTML that is used to make a webpage's design and structure.
