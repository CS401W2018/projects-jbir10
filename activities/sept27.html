List out the different ways to select an item.
ID and class tags

What is the difference between an ID and a Class?
ID is unique to one element, while class can apply to multiple elements.

What is the difference between a SPAN tag and a DIV tag?
A SPAN tag is for small parts of a page, while a DIV tag is for larger sections.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?
DIV tags can be used anywhere on the webpage, while HEADER, MAIN, FOOTER have fixed positions on the webpage.

What are different ways to select a color?
Using color names, hex codes, RGB or RGBA, HSV
