Learning CSS

Explain the difference between inline, embedded, and external CSS.
Inline CSS are added directly to the HTML elements using the style attribute.
Embedded CSS are written in the style tage inside the HTML file in the head section.
External CSS are kept in a seperate .css file linked to the HTML file using a link tag in the head section.

Explain why you should only use external CSS.
It makes changing styles easier throughout the website, keeps code neat.

What is a selector used for?
It is used to select the HTML element you want to style.
