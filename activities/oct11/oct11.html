What is the difference between a block and an inline tag? Give examples of each one.
A block tag takes up the full width available and starts on a new line (e.g., h1 tag). An inline tag only takes up as much width as necessary and does not start on a new line (e.g., img tag).

What is normal flow in HTML?
Normal flow in HTML is the default arrangement of elements in the order they appear, with block elements stacking vertically and inline elements flowing horizontally.

What does float do? Why do you need to clear the float?
The float property moves an element left or right, allowing text to wrap around it. Clearing the float prevents parent elements from collapsing and maintains proper layout.

What do absolute, fixed, and relative do?
Absolute positions an element based on its parent, fixed stays in the same place on the screen, and relative moves an element from its normal spot without leaving the flow.

How can you make nice navigation buttons for mobile devices?
