What is JavaScript? How did it get its name?
JavaScript is a programming language used to make web pages interactive. Its name was chosen to ride on Java's popularity.

What are the two ways you can run JavaScript?
You can run JavaScript in a browser or using a server-side environment like Node.js.

What was the most popular front-end js library?
jQuery was the most popular front-end JavaScript library.

What is the most popular front-end js library today?
React is the most popular front-end JavaScript library today.
